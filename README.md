# Python-LM
Python Machine Learning Project

## Descrizione
Questo progetto dimostra come creare un modello di Machine Learning con Python utilizzando scikit-learn.

## Installazione

```bash
pip install -r requirements.txt
```

## Utilizzo

```bash
python ml_model.py
```

## Struttura del Progetto

- `ml_model.py` - Script principale con il modello di classificazione
- `requirements.txt` - Dipendenze del progetto

## Tecnologie Utilizzate

- Python 3.x
- scikit-learn
- NumPy
- Pandas
